# Exploratory Data Analysis (EDA) on Retail Sales Data

## Project Overview

This project presents an exploratory data analysis of a retail sales dataset using Python. The analysis explores customer purchasing behaviour, sales trends, product performance, and relationships between key variables to generate actionable business insights that support data-driven decision-making.

## Objectives

- Inspect and clean the dataset.
- Perform descriptive statistical analysis.
- Analyze monthly and quarterly sales trends.
- Explore customer demographics.
- Evaluate product category performance.
- Identify relationships among numerical variables using correlation analysis.
- Generate actionable business recommendations based on the findings.

## Tools and Libraries Used

- Python
- Jupyter Notebook
- pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset Information

The dataset contains **1,000 retail sales transactions** with the following variables:

- Transaction ID
- Date
- Customer ID
- Gender
- Age
- Product Category
- Quantity
- Price per Unit
- Total Amount

Additional features created during the analysis include:

- Year
- Month
- Quarter
- Age Group

## Analysis Performed

The project includes:

- Data inspection
- Missing value and duplicate check
- Descriptive statistics
- Monthly sales trend analysis
- Quarterly sales trend analysis
- Customer age group analysis
- Gender distribution analysis
- Product category analysis
- Revenue by product category
- Correlation heatmap
- Box plot of transaction amount by product category
- Business recommendations

## Key Insights

- Electronics generated the highest total revenue.
- Clothing recorded the highest quantity sold.
- Customer age showed little relationship with spending behaviour.
- Product price had the strongest positive relationship with total transaction value.
- Sales performance varied across months and quarters, indicating seasonal trends.
- Most transactions were relatively small, while a few high-value purchases contributed significantly to overall revenue.

## Business Recommendations

- Increase investment in high-revenue product categories.
- Introduce promotions and cross-selling opportunities for Clothing products.
- Improve marketing strategies for Beauty products.
- Prepare inventory based on seasonal sales trends.
- Focus marketing efforts on purchasing behaviour rather than customer age.
